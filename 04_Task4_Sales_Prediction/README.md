# Sales Prediction Using Python

## Objective

The objective of this project is to predict product sales based on advertising expenditure across different marketing platforms.

## Dataset

Advertising Dataset

Features:

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget

Target Variable:

* Sales

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Data Analysis

* Loaded and explored the dataset
* Checked for missing values
* Visualized relationships between advertising budgets and sales
* Prepared data for machine learning

## Machine Learning Model

Linear Regression

## Results

The model was trained to predict sales using advertising expenditure data and achieved good prediction performance.

## Conclusion

Advertising expenditure has a significant impact on sales. The Linear Regression model successfully predicted sales based on marketing investments.
